Applicazioni Distribuite

Possiamo definire un’applicazione distribuita come un’applicazione costituita da due o più processi eseguiti, in parallelo, su macchine distinte connesse da una rete di comunicazione. I processi che costituiscono un’applicazione distribuita cooperano sfruttando i servizi forniti dalla rete di comunicazione.

Le applicazioni distribuite possono essere suddivise secondo tre livelli applicativi:

Frontend

Permetted di gestire la logica di presentazione e le modalità di interazione con l’utente.

Vantaggi nell'utilizzo di MEAN Stack

Linguaggio comune

Tutti i framework utilizzati, utilizzano un linguaggio comune: JavaScript.

Architettura MVC

supporta l’architettura MVC che separa la logica di presentazione dei dati dalla logica di business.

Competenze richieste

Lo sviluppo di un’applicazione MEAN richiede solo competenze JavaScript.

Prestazioni

NodeJS offre alte prestazioni per via della sua architettura non-blocking.

Documentazione

I siti web dei framework offrono robuste e dettagliate documentazioni

Costi

Tutte le tecnologie utilizzate in questa architettura sono gratuite e richiedono personale con meno competenze.

Funzionamento dell'architettura MEAN




Angular

Frontend
AngularJS è un framework web open source sviluppato da Google e dalla comunità di sviluppatori individuali, nato nel 2009 per affrontare le difficoltà incontrate nello sviluppo di applicazioni single-page.

Architettura MVC (Model View Controller), che separa la logica di presentazione dalla logica di business.

Nel 2016 compare una nuova release del framework completamente riscritta rispetto a quella precedente, con il nome di Angular.

AngularJS è scritto in JavaScript, mentre Angular in TypeScript.

NodeJS

Backend
NodeJS è un framework event-driven rilasciato nel 2009 da Joyent..

Dispone di molti moduli scritti in JavaScript, ma gli sviluppatori possono scrivere nuovi moduli utilizzando lo stesso linguaggio.

Il modello di networking non è quello dei processi concorrenti, ma I/O event-driven, poichè il flusso del programma è determinato dal verificarsi di eventi esterni.

Permette di scrivere codice JavaScript lato server.

ExpressJS

Middleware
ExpressJS è un framework di routing e middleware gratuito ed open-source rilasciato nel 2010, che mette a disposizione un set di componenti per la creazione di app single-page, multi-page e ibride..

Permette di semplificare lo sviluppo di applicazioni web per NodeJS.

Offre dei tools per semplificare la creazione di server HTTP.

MongoDB

DBMS
“MongoDB is a document database with the scalability and flexibility that you want with querying and indexing that you need”
- MongoDB -

MongoDB è un document database open-source e leader nel settore dei database NoSQL.

Scitto in C++, permette di creare database altamente scalabili e molto performanti.

Document-oriented: è importante capire il significato dei concetti collection e document.

Il database in sè è un contenitore di collection.

Una collection è un contenitore di document ed è paragonabile al concetto di tabella dei database relazionali.

Un document, è un set di coppie chiave-valore con schema dinamico. Si tratta di un documento JSON con schema dinamico (BSON), paragonabile alle tuple dei database relazionali.
